Bootstrap 5.3 Tooltips

Tooltips are small popups that appear when users hover over, focus on, or tap an element.

Hover or focus this to see a tooltip.

Tooltip on a link

Disabled button tooltip

Disabled elements do not fire mouse events, so wrap the disabled button in a focusable element.

Tooltip with custom trigger

How it works

  1. Add data-bs-toggle="tooltip" to the element.
  2. Add tooltip text with data-bs-title="...".
  3. Load Bootstrap's JavaScript bundle.
  4. Initialize tooltips with JavaScript.
